## Description

A simple Bookmark REST API with CRUD, Authentication and authorization using Nest js and Postgres with prisma.

## Followed Tutorial

Nest JS Course for Beginners -  [Video link](hhttps://youtu.be/GHTA143_b-s)
## Features

- Authentication
- Create Bookmarks
- Swagger UI Docs
- Bookmarks CRUD

## Tech Stack

- Nest JS
- Postgresql
- prisma
- swagger docs (documentation)



# Getting Started

```bash
$ git clone https://github.com/Yash-1511/nestlearning.git
```
## Installation

```bash
$ npm install
```
## configure your env

```
DB_URL=YOUR_DATABASE_URL
JWT_SECRET=YOUR_JWT_SECRET
```
## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
